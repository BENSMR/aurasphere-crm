### Step 2: Deploy App (15 minutes)
```
Option A: Vercel (Recommended)
  1. vercel deploy --prod
  2. Add environment variables
  3. Done!

  1. netlify deploy --prod --dir=build/web
  2. Add environment variables
  3. Done!

  1. firebase deploy --project your-project
  2. Add environment variables
  3. Done!
```
